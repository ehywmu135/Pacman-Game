#Pacman Game

A classic Pacman game clone built using C# and the .NET Framework.  
Developed as part of my computer science studies to practice object-oriented programming, event-driven logic, and game mechanics.

## Features
- Grid-based level layout
- Player-controlled Pacman movement
- Ghosts with basic pursuit logic
- Dot collection and scoring system
- Win and game over conditions
- Keyboard input handling

## Technologies Used
- Language: C++
- IDE: Visual Studio

## How to Run
1. Clone the repository or download the project ZIP.
2. Open the solution file (`.sln`) in Visual Studio.
3. Build the project and run.
